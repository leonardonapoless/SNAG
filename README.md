A macOS app for fetching and downloading multiple GitHub repositories from a GitHub profile at once.

I built this because I wanted an easier way to grab a bunch of repos from a user profile without having to clone each one manually.

## Features

- Fetch all public repositories for a given GitHub user
- Optionally skip forks
- Optionally separate by programming language
- Download specific repos or all of them at once
- Keeps track of recent profiles
- Pause, resume, or cancel active downloads
- Downloads directly to a custom folder inside your Downloads directory
- Built with SwiftUI
